REPORT API

Status Codes:
0 = sucess
1 = missing parameters
2 = invalid parameters
3 = unauthorize
4 = unavailable content
5 = server error

Status:
Success = SUCCESS
Error = ERROR

Http Codes:
200 = Success
201 = Created/ Updated/ Deleted
400 = Bad Request / Invalid Parameters or Input
401 = Not Authorized
404 = Not Found
500 = Internal Server Error

Methods Allowed:
GET
PUT
POST
DELETE
